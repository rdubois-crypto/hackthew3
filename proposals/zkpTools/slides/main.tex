\section{Zero Knowledge Proofs}

\begin{frame}
 \begin{center}
 \textcolor{yellow}{\fbox{Zero Kowledge Proofs}} 
 \end{center}
 
 

 \fbox{\parbox{\textwidth}{

 \begin{itemize} 
 \item Zero-knowledge proof is a method by which a prover can prove to a verifier that a given statement is true while the verifier doesn't learn information except the statement result.
 \item ZKP enable transaction hiding (Monero, ZCash), range proof, proof of membership, ZK-Rollups (Layer 2). 
 \item Most of building blocks for Privacy-Preserving and ZKP rely on the use of {\bf Pairings over Elliptic curves} (Zk-snarks, Commitments, Efficient signatures)
 \item Ledger direct examples : Linkable Signatures for Anonymous airdrop ({\bf device ID}), Ring signature ({\bf endorsement}), Strongly Verifiable SS ({\bf Protect}), {\bf Decentralized Identifiers}.
 \end{itemize}
 }}
    
\end{frame}
